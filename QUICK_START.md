## Instant Setup (Windows)
1. Double-click `setup.bat`
2. Wait for installation to complete
3. Server will start automatically
4. Open browser to `http://localhost:3000`

## Instant Setup (Mac/Linux)
1. Run `./setup.sh` in terminal
2. Wait for installation to complete
3. Server will start automatically
4. Open browser to `http://localhost:3000`

## Manual Setup
```bash
npm install
npm start
```

## What You'll See

### Student Dashboard
- 14-day color-coded calendar showing stress levels
- AI recommendations for managing workload
- Upcoming deadlines with difficulty ratings
- Ability to add personal events

### Professor Dashboard
- Class load overview with weekly graphs
- Conflict detection for overlapping deadlines
- AI suggestions for optimal scheduling
- Student risk monitoring

### Admin Dashboard
- System-wide statistics and department comparisons
- Configuration management for load thresholds
- Professor activity monitoring
- System alerts and notifications

## Need Help?
- Check that Node.js is installed: `node --version`
- Ensure port 3000 is available
- Try refreshing the browser if login doesn't work
- Check browser console for any JavaScript errors
